# @gonnect/rust-kgdb - Basic Usage Example

This example demonstrates how customers will use the package.

## Installation

```bash
npm install
```

## Run Example

```bash
npm test
```

## What This Example Covers

1. ✅ **Basic Triple Store** - Loading and querying RDF data
2. ✅ **RDF* (RDF-star)** - Quoted triples for metadata
3. ✅ **Hypergraph Operations** - N-ary relationships with hyperedges
4. ✅ **RDFS Reasoning** - Schema inference
5. ✅ **Datalog Reasoning** - Custom logic rules
6. ✅ **Performance Features** - Zero-copy, SIMD, parallel execution

## Expected Output

The example runs successfully and demonstrates all major features without requiring actual binary integration (documentation test).
