rust-kgdb (Our Product):
  - RDF/SPARQL 1.1 + Hypergraph database
  - Native Datalog inference engine
  - Mobile-first (iOS/Android via FFI)
  - Zero-copy architecture
  - 100% W3C compliant
  
TypeDB (Commercial - Vaticle):
  - Polymorphic database with type system
  - TypeQL query language (proprietary)
  - Java-based (JVM required)
  - Strong schema enforcement
  - Commercial + Open Source editions

IndraDB (Rust Graph DB):
  - Property graph database (not RDF)
  - No SPARQL support
  - Rust-based
  - Focused on graph traversals
  - Apache 2.0 license

Terminusdb (Knowledge Graph):
  - RDF + SPARQL + Datalog
  - Prolog-based (SWI-Prolog)
  - Git-like versioning
  - Commercial focus
  - JavaScript/Rust hybrid
